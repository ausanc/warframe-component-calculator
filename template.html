Warframe Component Cost Calculator
Tool version 1.1 for Warframe version 29.6 Orphix Venom

This page uses cookies to store which items you have selected.

This calculator lists the required components to build the selected items. Only items which are built in the foundry are listed, so things like weapons purchased from Baro are not listed here. Please note that parts such as a Warframe's chassis / systems / neuroptics are not listed individually, instead the cost of each of these sub-components is included in the parent item's costs. This also applies to weapons which use another weapon in their recipe, but excludes Forma, Fieldrons, Mutagen Masses, Detonite Injectors, and refined minerals and gems. This tool is intended to be used by scrolling through your codex (sort by rank) and selecting items which have not yet been mastered / made. Note that this is why arch-guns are listed under Primary, arch-melee under Melee, and archwings under Warframes; they are sorted like this in the codex.
